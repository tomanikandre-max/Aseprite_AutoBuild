# Para que serve este tutorial
Caso tenha dificuldades para usar o AsepriteAutoBuild este tutorial mostra o passo a passo de como usalo.
# Abrindo o programa
* Baixe o [Relase aqui no Github](https://github.com/tomanikandre-max/AsepriteAutoBuild/releases/download/1.0.2/AsepriteAutoBuild.exe) ou se estiver desconfiado(a) compile seguindo o [Instruções.md](https://github.com/tomanikandre-max/AsepriteAutoBuild/blob/main/Instru%C3%A7%C3%B5es.md).
* Caso você baixe o relase pronto ele vai vir em **SFX (SelF-eXtracting)**, não é recomendado usar no disco, e nunca use na área de trabalho ou outros locais senssíveis, porque o programa adiciona a pasta as excluôes do Windows defender para evitar erros de compilação por falsos positivos, o ideal é por em uma pasta isolada, caso tenha compilado ele vai estar na pasta de seu projeto.
# Configurando a instalação
* Ele vai abrir, selecione a sua língua, se você for brasileiro(a), português(a), ou qualquer país que fale esta língua, selecione **Português**, caso contrário selecione **Inglês**.
* Configure o local de instalação, ou deixe em **C:\Aseprite**
* Ele vai instalar, se der erro, você pode verificar se o disco **C/D** está cheio ou o caminho que você escreveu está errado, caso não, você tem todo o direito de comentar aqui no Github, isso me ajuda muito à resolver falhas e fazer novos testes.
# Despedida
Funcionou!? Você acabou de compilar um programa difícil do zero, parabéns, obrigado por usar o ****AsepriteAutoBuild!****
